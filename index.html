Game Features

Everything you need for an epic text adventure experience

🎮

5 Character Slots

Create multiple heroes with different builds. Perfect for trying new strategies or sharing with friends.

⚔️

Turn-Based Combat

Strategic battles with health bars, damage calculations, status effects, and coordinated party strikes.

🗺️

Open World

Explore villages, forests, deserts, arctic tundras, glitch zones, and more. Your choices shape the story.

👥

Party Multiplayer

Team up with friends for Monster Hunt, Void Expedition, Frost Pact, and Sun Covenant quests.

🗣️

Chat & Social

Room chat, whispers, party invites, and notifications. Stay connected while you play.

🤖

AI Wise Man

Get contextual hints and lore from Gemini or Cloudflare AI. Your personal adventure guide.

💾

Local Saves

SQLite persistence keeps your progress safe. Works fully offline without any cloud dependencies.

📈

Progressive Leveling

Earn XP, gold, and skill points. Unlock new abilities and equipment as you grow stronger.

🏆

Side Quests

Collect the Ruins Token, Moon Charm, and Star Map for the legendary Triptych Blessing.

Ready to Begin Your Adventure?

Install in seconds with Homebrew, or download a binary directly. Works on macOS, Linux, and Windows.

Download Latest Release
Install Commands 
# macOS / Linux (Homebrew)
brew tap MaizerGomes/magicadvanture
brew install magicadventure

# Or run directly
go run .

# Build from source
git clone https://github.com/MaizerGomes/magicadvanture
cd magicadventure
go build -o magicadventure .
