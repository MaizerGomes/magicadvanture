Turn-Based Combat

Strategic battles with visible health bars, damage calculations, and status effects. Freeze enemies, poison them, burn for damage-over-time, or stun for tactical advantage. Party members can coordinate strikes for massive damage.

  • ✓ Health Bars — See enemy HP in real-time
  • ✓ Status Effects — Freeze, Poison, Burn, Stun
  • ✓ Coordinated Strikes — Party attacks deal bonus damage
  • ✓ Loot Drops — Gold, XP, and equipment

Install via Homebrew 
# macOS / Linux
brew tap MaizerGomes/magicadvanture
brew install magicadventure

# Or run from source
go run .

# Build from git
git clone https://github.com/MaizerGomes/magicadvanture
cd magicadventure
go build -o magicadventure .
